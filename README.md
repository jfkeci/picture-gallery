## Local setup


### App

Install dependencies
```bash
npm install
```

Serve app localy
```bash
npm run serve
```




### API

```bash
npm install
```

Serve app localy
```bash
npm start
```

When you run the API locally you can se basic example of swagger api documentation at:
http://localhost:5001/api-docs
